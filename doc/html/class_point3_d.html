Detailed Description

A 3D point representation.

This geometrical abstraction represent a point in space. Values are assumed to be in meters.

double Point3D::squaredDistanceTo ( Point3D *  point)

Calculates the square of the distance to another point.

This is faster than calculating the actual distance, and end in the same results when what is intended is to calculate relative distance (i.e. calculate the closest point to another point).
