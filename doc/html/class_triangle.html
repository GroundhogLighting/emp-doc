bool Triangle::testPoint ( Point3D *  p,
int *  code 
)

Tests if a Point3D is inside the Triangle.

This function tests if a Triangle contains a certain Point3D, considering the edges as "inside" the Triangle.

The "Code" generated will vary according to the position of the Point3D in the Triangle. A code of -1 is returned when the Point3D is not in the Triangle; 0,1,2 when the Point3D is on vertices 0,1 or 2, respectively; 3, 4 and 5 mean that the Point3D is on edges 0,1 and 2, respectively; and a code of 6 means that the Point3D is inside the Triangle.
