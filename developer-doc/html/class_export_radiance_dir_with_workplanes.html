Detailed Description

Exports not only the geometry, but the workplanes as well.

Implementing the exporting prcess in this way allows multicore exporting... One thread is used for exporting geometry, and many others to triangulate and write workplanes.

The triangulation and writing of the Workplanes depend on the exportation of the geometry
