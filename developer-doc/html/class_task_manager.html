Detailed Description

The TaskManager is an object that performs the calculation in an efficient way.

The TaskManager intends to perform all the calculations needed by the GroundhogModel or user in a safer, efficient and easier way. In order to do that, certain characteristics have been added to this object:

  • It allows developers and users to define only the Task they want to solve (i.e. Calculate Daylight Autonomy) and it will define what needs to be done in between (which is called Expand)
  • During expansion, the TaskManager will only add the tasks that are not redundant (i.e. calculating twice the Daylight Autonomy of a workplane)
